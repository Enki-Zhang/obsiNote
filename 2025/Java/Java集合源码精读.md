集合框架中，两个重要的接口分别是Collection和Map;

### Collection

其中Collection接口中分别被List、Set 、Queue 三个子接口继承，他们同时也代表了集合的不同特性；
1.  List，代表了有序、可重复集合
	1. 实现List接口的对象有ArrayList、LinkedList
2. Set，代表了无序不可重复的集合
	1. 实现Set接口的对象有 HashSet、TreeSet
3. Queue表示队列





### Map

Map接口表示键值对形式的集合，
